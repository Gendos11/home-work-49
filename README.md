# User Profile Fetcher

Small React app that fetches a user profile from a remote API and displays loading, success, and error states. Includes unit tests with mocked API responses.

## Setup

```bash
npm install
```

## Run the app

```bash
npm run dev
```

## Run tests

```bash
npm run test
```
